# Slicing Acarra App 

Slicing screen appointment of Acarra app as shown in the figma.

## Run Locally

Clone the project

```bash
    git clone https://github.com/firman19/upwork-project-1.git
```

Go to the project directory

```bash
    cd upwork-project-1
```

Install dependencies

```bash
    npm install
```

Start the app

```bash
    npm start
```

## Tech Stack
HTML5, CSS3
